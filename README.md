## Overview
PH Green Tree is an interactive web application focused on ecological awareness and tree management.  
Users can explore different tree species, view details about them, and simulate planting trees in a virtual environment.  
This project helps promote environmental education and sustainable practices.

## Live Demo
ðŸ”— [Live Site on Vercel](https://ph-green-tree.vercel.app)  

## Technology Stack
- **React 18+** (Frontend framework)  
- **Vite** (Development and build tool)  
- **CSS** (Styling)  

## Main Features
- Browse and explore various tree species  
- Detailed information about each tree (age, type, water requirements, shade, etc.)  
- Virtual planting simulation for user interaction  
- Fully responsive design for mobile and desktop  

## Dependencies
- react  
- react-dom  
- vite
